chore: exclude CMakeFiles from source style checks

Added --exclude-dir=CMakeFiles to grep commands in check-source-style.sh
to prevent false positives from CMake-generated code.

Changes:
- Exclude CMakeFiles from non-ASCII character check
- Exclude CMakeFiles from block comment check

This eliminates spurious warnings about block comments in
tests/CMakeFiles/*/CompilerIdC*.c files which are auto-generated by CMake.
